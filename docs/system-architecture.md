## 1. Architecture Overview

### 1.1 Design Principles

1. **Microservices-ready** вҖ” Start as modular monolith, split when needed
2. **Event-driven** вҖ” Async processing for non-blocking operations
3. **Real-time first** вҖ” WebSocket for voice, SSE for text updates
4. **Cost-optimized** вҖ” Cache aggressively, batch API calls, use cheaper models where possible
5. **Vietnam-optimized** вҖ” Low latency for VN users, local payment gateways, Vietnamese language support
